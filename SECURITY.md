# Security Policy

## Reporting Vulnerabilities
Please report security issues to security@therayyanawaz.co.in.  
**Do not disclose publicly** until we release a patch.

## Supported Versions
| Version | Supported          -   |
|---------|------------------------|
| 2.0.x   | ✅ Active support  -   |
| 1.0.x   | ⚠️ Security fixes only |
